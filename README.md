## プロダクト説明

<!-- プロダクトの説明を入力 -->

ユーザーが商品名、賞味期限といったような食品の情報を登録することで、その食品の管理が容易に行えるようになる。
また、食品はユーザーが決めたカテゴリごとに分類して管理することができ、賞味期限が近づくと赤く表示されるという機能もある。
